# CONTEXT

I am building a public-interest dataset on institutional ownership of
single-family homes in Travis County, Texas (Austin). I have scraped
the county clerk's deed records and grouped corporate entities into
buckets by name pattern. The 9 buckets below need to be linked
to their ultimate corporate parent — the publicly-traded REIT, PE firm,
fund, or operating company that controls them. Each bucket represents a
group of LLCs/LPs that appear to be coordinated based on naming and
counterparty patterns in deed records.

# RULES

1. For each bucket, identify the ultimate corporate parent.
2. CITE A SOURCE for every answer. Acceptable sources, in order of
   preference:
     - SEC EDGAR filings (10-K, 10-Q, S-1, ABS-EE, prospectus)
     - Company press release
     - Major news outlet (WSJ, Bloomberg, Reuters, FT, NYT)
     - Crunchbase, Pitchbook, or PE firm portfolio page (lower confidence)
3. If you cannot find a citable source, write 'UNKNOWN' for the parent
   and 'no_public_source' for the URL. DO NOT GUESS based on naming
   similarity. A wrong answer is worse than no answer.
4. For securitization SPVs (e.g., 'BORROWER 2021-1' style names), the
   sponsor is named in the EDGAR ABS-EE filing — that is the parent.
5. Confidence rating: HIGH (SEC filing or press release), MEDIUM (news
   article or PE portfolio page), LOW (only Crunchbase / unconfirmed).

# OUTPUT FORMAT

Return a single markdown table with these columns and nothing else
after the table (no commentary, no preamble):

| bucket | parent | confidence | source_url | notes |
|---|---|---|---|---|
| EXAMPLE BUCKET | Example Parent Inc | HIGH | https://www.sec.gov/... | brief context |

- One row per bucket, in the same order I list them below.
- If a bucket has multiple plausible parents, pick the most likely and
  list the alternatives in the notes column.
- Keep notes under 25 words.
